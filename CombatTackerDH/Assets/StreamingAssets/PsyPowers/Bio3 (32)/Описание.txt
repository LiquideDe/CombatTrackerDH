Психотест: W–30 Дальность: Сам
Эффект: Псайкер отращивает биологические стрелковое оружие из своего тела. Псайкер может отращивать это оружие, трансформируя руку, торс или голову, для чего выбранная часть тела не должна быть покрыта броней (кроме как с модификацией Вторая Кожа), хотя открытые шлемы не мешают трансформации головы. Псайкер считается вооруженным пистолетом с одним из следующих профилей:
Иглострел. Дальность PR×5м, RoF О/3/6, урон 1к10+PR Реж, пробитие PR, Калечащее (2), Разброс
Кислота Дальность PR×2,5м, RoF S/–/–, урон 1к10+2×PR Э, пробитие PR, Разъедающее (2), Распыляющее 
Копьемет Дальность PR×10м, RoF S/–/–, урон 1к10+2×PR Реж, пробитие 2×PR, Меткое, Проникающее, только из руки.
Хлыст Дальность PR×5м, RoF S/–/–, урон 1к10+2×PR Реж, пробитие PR, Точное, Терзающее, только из руки, может использовать НР вместо НС, и его попадания вызывают Контакт Плоти. В рукопашной он получает свойство Гибкое.
Оружие из руки трансформирует ладонь и предплечье, делая их непригодными для других задач. Оружие из головы и торса работает по правилам наплечного оружия, но головное не дает персонажу говорить, а из торса получает +1к10 к урону и свойство Перезарядка.
